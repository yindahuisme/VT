### 通用
```
1.侧边栏
    1.素材文件
    2.素材
    3.轨道
    4.特效
    5.模板创作
    6.设置
2.项目头
    1.标题展示
    2.保存按钮
    3.加载项目按钮
```

### 一.素材文件
```
1.素材文件展示区
    1.sql下拉框
    2.sql下拉框选项sql配置按钮
    3.搜索按钮
    4.素材文件展示框
        1.素材文件：左键选中，可右键重命名，删除，打点（入点，出点，事件点），如果是图片，可右键生成图片视频素材文件，如果是音频，可右键识别为文字视频素材文件
        2.空音频，空视频：占位素材文件
    5.刷新按钮：扫描素材文件根路径所有素材文件，新增素材文件
2.素材文件信息展示区
    1.素材文件预览：可简单播放素材
    2.素材文件信息
        1.文件大小
        2.时长
        3.生成时间
    3.生成的素材列表（所有项目）
```
### 二.素材
```
1.素材展示区
    1.sql下拉框
    2.sql下拉框sql配置按钮
    3.搜索按钮
    4.素材展示框
        1.素材：左键选中，可右键应用到轨道，删除（没有项目使用该素材）
            1.应用到轨道：弹出调整框，自动匹配点位后，可手动调整卡点
2.素材信息展示区
    1.素材预览：选中时可播放素材，编辑打点
    2.素材信息
        1.事件点数
        2.时长
        3.生成时间
        4.被几个项目使用
3.轨道匹配区
    1.轨道下拉框
    2.轨道匹配选择区：展示轨道当前素材填充情况，选择轨道区域


```
### 三.轨道
```
1.轨道区
    1.音，视频轨道
        1.初始一个主轨道
        2.轨道单位区域：左键选择，左键滑动框选，右键删除素材，在上（下）方添加轨道，设为匹配区域，添加特效，添加过渡效果
    3.主音频/主视频切换按钮：当各轨道中素材为空时可切换
2.特效列表区：当素材选中时展示其应用的特效
```
### 四.特效
```
1.特效分类区
    1.创建分类按钮：右键创建特效
        1.特效：左键选择，右键可删除，移动，拷贝
2.特效参数区：当选择特效时，展示，编辑其参数，生成特效预设
    1.特效名
        2.刷新按钮：通过特效名匹配pr特效
    2.参数区
    3.保存，取消按钮
```
### 五.模板创作
```
1.模板分类区
    1.创建分类按钮：右键创建模板
        1.模板：左键选择，右键可删除，移动
2.模板参数区：当选择模板时，展示，编辑其参数，生成模板预设
    1.项目名：不能重复，作为最终项目文件生成目录
    2.sql区：通过sql匹配素材，特效相关标签生成自动补全轨道
    3.保存，取消，生成按钮：生成时轨道必须没有素材
```
